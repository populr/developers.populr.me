developers.populr.me
====================

Public Developer documentation

#Build

`npm run metalsmith`

#Development Server

`npm run develop`

#Deploy

`npm run deploy`

Use triple backticks in documentation to escape html entities.

```
    ```
    <p>Hello world!</p>
    ```
    
    becomes
    
    &lt;p&gt;Hello world!&lt;/p&gt;
```
